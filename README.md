# Arduino
This repository contains projects developed using Arduino IDE. Arduino is a hardware/software platform that allows learners to program a microcontroller on their board with instructions using C, C++, or Arduino Programming Language (Arduino API). These programs read inputs, like from a sensor, and turn them into outputs, like turning on an LED.

My projects are sorted by the main component that I was learning to use at that point of time or cumulative projects designed with all the components in mind.

Components:
Arduino MEGA controller board
Breadboard
Breadboard wires
Resistors
LEDS
Buzzers
Photoresistors
Buttons
Servo motor
Capacitators
Switches
LCD module
Sound sensor
Ultrasonic sensor
Water level detection sensor
Digit display
